Contexte

Une entreprise a besoin d'une application pour faire de la gestion de tickets. Un ticket est-une série d'informations qui permet de décrire un problème technique que peut rencontrer un utilisateur. Un ticket peut-être crée, modifié, supprimé.

Objectif

L'objectif de la SAE23 est de prendre en main plusieurs outils dans le développement web coté BACKEND, entre autre :

  • NODE JS
  • Express
  • Mongo DB

Ma mission

Ma mission était de proposer une application Web fonctionnelle afin de faire de la gestion de ticket avec les technologies proposées.
